# 환영합니다

Canvas 프로젝트에 합류하신 것을 환영합니다. 이 문서는 신규 팀원이 **첫 주 안에** 개발 환경을 구축하고, 코드베이스의 핵심 개념을 이해할 수 있도록 돕습니다.

## Canvas란?

**프로젝트 의사결정을 돕는 시각적 프로토타입 에디터**입니다.

기획·디자인·개발 단계에서 UI 흐름, 화면 구성, 정보 구조를 빠르게 스케치하고 공유할 수 있습니다. Figma와 달리 **개발자 친화적인 데이터 모델**과 **확장 가능한 노드 시스템**을 갖추고 있어, 향후 협업·내보내기·자동화 기능으로 확장하기 쉽습니다.

## 핵심 가치

- **빠른 시각화** — 캔버스에 노드를 배치해 아이디어를 즉시 공유
- **일관된 에디터 UX** — 커맨드·단축키 패턴으로 조작 학습 비용 최소화
- **선언적 확장** — 새 노드 타입을 `NodeDefinition`으로 등록해 추가
- **모노레포 협업** — web / api / packages를 한 저장소에서 관리
