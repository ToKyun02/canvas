# 캔버스 사용법

Canvas 에디터의 기본 조작 방법입니다.

## 접속

1. http://localhost:3001 접속
2. **Go to Canvas** 클릭
3. 캔버스 에디터 화면 진입

## 도구

| 도구   | 단축키 | 설명                      |
| ------ | ------ | ------------------------- |
| 이동   | `V`    | 노드 선택·드래그·리사이즈 |
| 텍스트 | `T`    | 클릭하여 텍스트 박스 배치 |

텍스트 노드 배치 후 자동으로 편집 모드에 진입하며, 배치가 완료되면 이동 도구로 전환됩니다.

## 선택 & 편집

| 동작      | 단축키                 |
| --------- | ---------------------- |
| 전체 선택 | `Mod+A`                |
| 선택 해제 | `Escape`               |
| 삭제      | `Delete` / `Backspace` |

선택된 노드는 우측 Properties Sidebar에서 확인할 수 있습니다. 현재 사이드바는 UI 스캐폴딩 단계이며, 노드별 속성 편집은 아직 연결되지 않았습니다.

## 뷰 조작

| 동작          | 단축키  |
| ------------- | ------- |
| 확대          | `Mod+=` |
| 축소          | `Mod+-` |
| 화면에 맞추기 | `Mod+0` |

마우스 휠로도 줌/팬이 가능합니다 (`useCanvasViewportWheel`).

## 에디터 UI

| 기능           | 단축키  | 설명                        |
| -------------- | ------- | --------------------------- |
| 속성 패널 토글 | `Mod+/` | 우측 Properties Sidebar     |
| 노드 라벨 토글 | `Mod+.` | 캔버스 위 노드 ID/타입 라벨 |
| 테마 전환      | UI 버튼 | 좌측 상단 다크/라이트 모드  |

## 상태 저장

에디터 상태는 **localStorage**에 자동 저장됩니다 (`canvas-app-v1` 키).

저장되는 항목:

- `nodes`, `nodeOrder` — 문서 데이터
- `isPropertiesSidebarOpen`, `isVisibleNodeLabels` — UI 설정

새로고침해도 작업 내용이 유지됩니다.

## 실행 취소 / 복원

| 동작      | 단축키        |
| --------- | ------------- |
| 실행 취소 | `Mod+Z`       |
| 복원      | `Mod+Shift+Z` |
